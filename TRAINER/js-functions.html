JS FUNCTIONS

1. Funktion ohne Argumente

Zweck: Funktion Anweisungen ausführen zu lassen, für die man keine weitere Daten (Variablen) braucht.
        function myFunction() {
        // eine JS Anweisung
    }

Beispiel. Szenario: Beim Klick auf Button wird Datum angezeigt

platzhalter

2. Funktion mit Argumenten

Zweck: Daten in die Funktion zu übergen, die von der Funktion iwie verwendet oder bearbeitet werden müssen
        Weg: in den runden Klammern eine Variable anlegen ohne jegliche Schlüsselwörter (var, let, const)
        function myFunction(argName) {
            // eine JS Anweisung mit argName
        }

Beispiel 1. Szenario: Beim Input in ein Inputfeld wird dieser Input angezeigt

3. Funktion mit Rückgabe (return)

        Begriffe, die man hier braucht:
        Ausdruck (expression) => liefern Werte
        Anweisung (statement) => verwenden Werte und weisen diese zu anderen Platzhaltern zu.
