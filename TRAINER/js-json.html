JSON

        JavaScript Object Notation
        ein String in der Syntax von JavaScript
        Dieses String kann in Objekte / Arrays / Primitiven umgewandelt werden (deserialisiert)
        Beispiel:
        [
            {
                "userId": 1,
                "id": 1,
                "title": "delectus aut autem",
                "completed": false
            },
            {
                "userId": 1,
                "id": 2,
                "title": "quis ut nam facilis et officia qui",
                "completed": false
            }
        ]
